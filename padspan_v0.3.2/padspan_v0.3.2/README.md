# PadSpan Suite v0.3.2

This package contains two tracks:

1. **padspanHA** — Home Assistant integration (HACS-ready)
2. **padspan-enterprise** — standalone hub + receivers + mobile/web app architecture starter

## What's included in v0.3.2
- Sidebar navigation refactor with stable toggle/expand behavior.
- Route-aware active nav highlighting.
- Persisted collapsed/expanded sidebar state (localStorage).
- Basic map calibration scaffolding and BLE observation API stubs.
- Starter architecture for enterprise track (hub, receivers, mobile app specs, admin console).
