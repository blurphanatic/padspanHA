PadSpan HA v0.3.17 (local-first + sidebar restore)

Highlights:
- Config flow hardened and simplified (scan interval only)
- Local-first setup (no API key required; cloud disabled by default)
- Sidebar registration retained and preflighted in async_setup_entry
- Full sidebar feature set retained + expanded:
  - collapse/expand toggle
  - layout modes (fixed/overlay/inset)
  - width modes (full/mini-rail)
  - density (cozy/compact/dense)
  - surface styles (solid/glass/segmented)
  - keyboard shortcuts: [, Shift+[ , Esc
  - many menu variations for testing
- Room checklist + dynamic tag filtering (ALL selected rooms or ANY selected rooms)
- Auto diagnostics retained

Install clean:
1) Stop HA
2) Remove:
   /config/custom_components/padspan
   /config/custom_components/padspan_ha
3) Copy custom_components/padspan_ha from this zip to /config/custom_components/
4) Start HA and hard refresh (Ctrl+F5)
