Clean install (important):
1) Stop HA
2) Remove:
   /config/custom_components/padspan
   /config/custom_components/padspan_ha
3) Copy custom_components/padspan_ha from this zip to /config/custom_components/
4) Start HA
5) Hard refresh browser (Ctrl+F5)

If Configure still errors:
- Enable debug logs:
  logger:
    default: info
    logs:
      custom_components.padspan_ha: debug
      homeassistant.config_entries: debug
- Then open Configure and copy the traceback.
