PadSpan HA v0.3.13 install (single domain)

1) Stop Home Assistant
2) Remove old folders:
   - /config/custom_components/padspan
   - /config/custom_components/padspan_ha
3) Copy this package's /custom_components/padspan_ha into /config/custom_components/
4) Start Home Assistant
5) Hard refresh browser (Ctrl+F5)

If duplicate integration rows still appear:
- Settings -> Devices & Services -> Integrations
- Delete disabled legacy entry

Optional logger for troubleshooting:
logger:
  default: info
  logs:
    custom_components.padspan_ha: debug
    homeassistant.loader: debug
    homeassistant.config_entries: debug
