# Home Assistant Brands assets (PadSpan)

Integration card icons/logos in Home Assistant come from:
https://github.com/home-assistant/brands

Included here:
- docs/brands/custom_integrations/padspan_ha/icon.png
- docs/brands/custom_integrations/padspan_ha/icon@2x.png
- docs/brands/custom_integrations/padspan_ha/logo.png
- docs/brands/custom_integrations/padspan_ha/logo@2x.png

To make the integration card show your custom logo, submit these files in a PR under
`custom_integrations/padspan_ha/` in the brands repository.
